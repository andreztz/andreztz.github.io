Remover arquivos e diretórios do github

Os comandos a seguir remove arquivos ou diretórios do sistema de arquivo do repositório remoto git, mas não no sistema de arquivo local.

$ git rm --cached ARQUIVO

ou

$ $ git rm --cached -R DIRETORIO/
